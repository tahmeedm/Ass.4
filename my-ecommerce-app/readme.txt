In order to run the front end and backend concurrently, you will need to first open a terminal and run the following commands:
    1. Navigate to Backend directory: cd my-ecommerce-app/Backend
    2. Run python app: python app.py

Then, open another terminal at the same time and run the following commands: 
    1. Navigate to my-ecommerce-app directory: cd my-ecommerce-app
    2. Run React App: npm start
